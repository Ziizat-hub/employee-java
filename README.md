# Employee Management App

A Full Stack REST API built with Java Spring Boot and MySQL.

## Tech Stack
- Java 21
- Spring Boot
- Spring Security
- MySQL Database
- Maven

## Features
- Employee Management (CRUD)
- User Authentication
- Email Notifications
- REST API Endpoints

## Setup
1. Clone the repo
2. Configure your database in application.properties
3. Run with: ./mvnw spring-boot:run
